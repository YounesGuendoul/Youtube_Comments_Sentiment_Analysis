# Youtube_Comments_Sentiment_Analysis
This project conducts sentiment analysis on YouTube comments by extracting them using youtube api v3 and applying the emoRoberta model by hugging face. This project is a website using flask it takes 3 inputs (url video - n°of comments - api_key) and it returns as an output the predominant sentiment of the comments.
